# Web3 Data Indexer

![Python](https://img.shields.io/badge/python-3.8+-blue)
![License](https://img.shields.io/badge/license-MIT-green)

一个用于索引和查询Web3链上数据的工具，支持以太坊、Polygon等网络。

## 功能
- 实时抓取链上交易数据
- 解析智能合约事件
- 提供REST API查询接口

## 快速开始

### 依赖安装
```bash
pip install -r requirements.txt  # 需先创建requirements.txt
